Aby uruchomić grę należy:

1. Pobrać repozytorium

2. W katalogu z pobranym repozytorium uruchomić terminal

3. Wpisać komendy

    make
  
    make install_client
  
4. Uruchomić serwer za pomocą komendy ./server

5. Uruchomić klienta za pomocą python3 klient.py
